# Companion-OS-v2

Your personal AI assistant for UNESCO Iraq operations and mission management.

## Features

- **Daily Briefings** - Morning summaries at 9:30 AM Baghdad
- **Mission Reports** - Draft reports from your notes
- **Team Admin** - HR, IT, Procurement, Finance reminders
- **Telegram Integration** - Receive updates on the go

## Getting Started

1. Pair Telegram bot: @JC_iso_bot
2. Send mission notes for report drafting
3. Set up daily briefings

## Files

- `USER.md` - Your profile and preferences
- `SOUL.md` - AI assistant persona
- `MEMORY.md` - Long-term memory
- `memory/` - Daily notes
- `second-brain/templates/` - Mission report templates
